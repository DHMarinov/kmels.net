Relaciones de Equivalencia


Idea: Considerar varios objetos matemáticos como esencialmente el mismo.


Ejemplo 1: En el plano cartesiano, en donde pueden haber transformaciones como rotación, alargamiento, etc.


Ejemplo 2: Congruencia módulo 2 en Z (enteros)


4 mod 2 = 0

5 mod 2 = 1


Relaciones de equivalencia, son:

  • Reflexiva: xRx, x~x .
    • La relación ">" no es reflexiva.
  • Simétrica: xRy => yRx
    • La relación "mayor o igual" no es simétrica.
  • Transitiva: xRy ^ yRz => xRz


Relación de Orden: Cumple con Reflexividad y Transitividad.


Recomendación de Libro: Halmos, "Naive Set Theory".


Ejemplo 3:

Congruencia módulo 3.


Zmod3 equivalentes al 0 = { ... , -6,-3,0,3,6... } = [0]

Zmod3 equivalentes al 1 = { ... , -5,-2,1,4,7,...} = [1]

Zmod3 equivalentes al 2 = { ..., -4, -1, 2, 5, 8... } = [2]

[0],[1] y [2] son Clases de equivalencia
